# Code Style Guide

## Language

Use US English for everything except maybe future translation specific strings.

## Encoding

Use UTF-8 as source file encoding.

## Code Layout

### Indentation

Use spaces only, with 4 spaces per indentation level.

### Blank Lines

Seperate object definitions with 2 blank lines.

Use a single blank line between an object definition and alterations of the object.

### Trailing Whitespace

Do not include trailing whitespace on any lines.
